Linux高并发通讯框架阅读参考                                                   
=======================

**Sim_server项目实现的功能**<br>
 * 使用配置文件调配程序
 * 日志记录和打印
 * 守护进程后台运行
 * 多进程并发
 * 多线程同步
 * 线程池
 * 连接池
 * 自定义通讯包及封包、解包<br>
 以上功能经测试可用
